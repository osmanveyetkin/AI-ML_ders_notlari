1. Bulanık Mantık Nedir?

Bulanık Mantık, bilgisayarların "proje karmaşıklığı **oldukça yüksek**" veya "ekip deneyimi **fena değil**" gibi insani ve yoruma dayalı kavramları anlamasını sağlayan bir yaklaşımdır. Kesin 1/0'lar yerine, 0 ile 1 arasında değişen **üyelik dereceleri** kullanarak "gri alanları" yönetir.

B. Kurallar ve Sonuç

4. Python Kodu ve Simülasyonu

Simülatörde gördüğümüz mantığı Python'da `scikit-fuzzy` kütüphanesi ile kodlayabiliriz.

Kurulum

pip install -U scikit-fuzzy

Örnek Kod: Proje Risk Analizi

import numpy as np
import skfuzzy as fuzz
from skfuzzy import control as ctrl

# Girdi ve Çıktı Değişkenlerini Tanımla
size = ctrl.Antecedent(np.arange(0, 101, 1), 'size')
complexity = ctrl.Antecedent(np.arange(0, 101, 1), 'complexity')
experience = ctrl.Antecedent(np.arange(0, 101, 1), 'experience')
risk = ctrl.Consequent(np.arange(0, 101, 1), 'risk')

# Üyelik Fonksiyonları
size['küçük'] = fuzz.trapmf(size.universe, [0, 0, 25, 50])
size['orta'] = fuzz.trimf(size.universe, [25, 50, 75])
size['büyük'] = fuzz.trapmf(size.universe, [50, 75, 100, 100])
complexity.automf(names=['düşük', 'orta', 'yüksek'])
experience['deneyimsiz'] = fuzz.trapmf(experience.universe, [0, 0, 25, 50])
experience['ortalama'] = fuzz.trimf(experience.universe, [25, 50, 75])
experience['deneyimli'] = fuzz.trapmf(experience.universe, [50, 75, 100, 100])
risk.automf(names=['düşük', 'orta', 'yüksek'])

# Kuralları Tanımla
rule1 = ctrl.Rule(experience['deneyimli'] | size['küçük'], risk['düşük'])
rule2 = ctrl.Rule(experience['ortalama'], risk['orta'])
rule3 = ctrl.Rule(size['büyük'] & complexity['yüksek'], risk['yüksek'])
rule4 = ctrl.Rule(experience['deneyimsiz'], risk['yüksek'])

# Kontrol Sistemini Oluştur
risk_ctrl = ctrl.ControlSystem([rule1, rule2, rule3, rule4])
risk_sim = ctrl.ControlSystemSimulation(risk_ctrl)

# --- Python Çıktı Simülasyonu ---
# Girdi değerlerini aşağıdaki alana girerek bu kodun çıktısını simüle edin
risk_sim.input['size'] = 50
risk_sim.input['complexity'] = 50
risk_sim.input['experience'] = 50

risk_sim.compute()
print(f"Hesaplanan Risk: {risk_sim.output['risk']:.2f}%")
